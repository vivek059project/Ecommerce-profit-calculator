# E-commerce Profit Calculator

A responsive web application that helps online sellers calculate profit, expenses, profit margin, break-even price, and recommended selling price.

## Features

- Calculate Revenue
- Calculate Total Expenses
- Calculate Profit
- Profit Margin Percentage
- Break-even Selling Price
- Recommended Selling Price (30% Margin)
- Profit/Loss Status Indicator
- Profit Margin Health Bar
- Download Report Feature
- Dark Mode
- Mobile Responsive Design

## Built With

- HTML5
- CSS3
- JavaScript

## Project Purpose

I frequently compare product costs, shipping expenses, marketplace fees, and advertising costs when evaluating online selling opportunities. This tool helps quickly estimate profitability and pricing decisions.
